# 软件简介
Sail天气是一款界面美观、功能方便、简单易用的天气软件。

软件特色：
·Material Design设计，界面美观，查看数据方便，支持更换主题色、下拉刷新天气等；
·每天自动更换图片，保持新鲜感；
·查看未来多天的天气，提供有效的天气建议，未雨绸缪。
